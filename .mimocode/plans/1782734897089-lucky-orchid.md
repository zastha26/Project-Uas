# Dashboard User - Bank Sampah

## Overview
Membuat Dashboard User setelah login di path `/auth/dashboard` dengan menu: Profil, Setor Sampah, Riwayat, Saldo, Penarikan Saldo. Frontend-only dengan dummy data.

## Route Structure
```
app/auth/
  page.tsx          → Login/Register (existing, modify: redirect to /auth/dashboard on login)
  dashboard/
    page.tsx        → Dashboard layout + sidebar + all sections
```

## Implementation Steps

### Step 1: Modify `app/auth/page.tsx`
- Tambahkan state `isLoggedIn` dan `isRegistered`
- Saat login berhasil → redirect ke `/auth/dashboard` (gunakan `useRouter`)
- Tambahkan `onSubmit` handler pada form login

### Step 2: Create `app/auth/dashboard/page.tsx`
**Layout:**
- Sidebar kiri (w-64, collapsible on mobile via hamburger menu)
- Main content area (flex-1)
- Header dengan nama user + tombol logout

**Sidebar Navigation:**
- Logo "Bank Sampah"
- Menu items dengan icon SVG:
  1. Profil (user icon)
  2. Setor Sampah (recycle icon)
  3. Riwayat (clock icon)
  4. Saldo (wallet icon)
  5. Penarikan Saldo (bank-note icon)
- Tombol "Keluar" di bawah

### Step 3: Implement Each Section

**A. Profil**
- Card menampilkan: foto profil (placeholder), nama, email, telepon, alamat
- Tombol "Edit Profil" → toggle form edit (nama, telepon, alamat)
- Save button (simpan ke state local)

**B. Setor Sampah**
- Form fields:
  - Jenis Sampah: select dropdown (Plastik, Kertas, Kaca, Logam, Elektronik, Organik, Lainnya)
  - Berat (kg): number input
  - Deskripsi: textarea
  - Foto Sampah: file input (preview image)
  - Lokasi Penjemputan: text input
- Tombol "Kirim Setoran"
- Success alert setelah submit

**C. Riwayat**
- Tabel dengan kolom: No, Tanggal, Jenis Sampah, Berat (kg), Nominal (Rp)
- Filter/Search input di atas tabel
- Badge warna untuk jenis sampah

**D. Saldo**
- Card besar: "Saldo Anda" dengan nominal besar
- Ringkasan: Total Setoran, Total Penarikan
- Daftar transaksi terakhir (5 item)

**E. Penarikan Saldo**
- Card info: Saldo tersedia
- Form fields:
  - Jumlah Penarikan (Rp): number input
  - Nama Bank: select (BCA, Mandiri, BRI, BNI, lainnya)
  - Nomor Rekening: text input
  - Nama Pemilik Rekening: text input
- Tombol "Ajukan Penarikan"
- Warning jika nominal > saldo
